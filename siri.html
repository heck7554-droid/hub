🗣 Siri setup

One-time setup per phone, about 3 minutes per shortcut. You need the personal key a parent created in the app (Admin → Siri shortcuts).

The one shortcut recipe (used by all four)

  1. Open the Shortcuts app → tap + to make a new shortcut.
  2. Add the action Get Contents of URL.
  3. Paste the URL for the command you want (table below).
  4. Tap the little ▸ arrow on the action → Method: GET → Headers → add one: key Authorization, value Bearer hub_YOURKEY (the word Bearer, a space, then the whole key).
  5. Add the action Get Dictionary Value → get value for key → type text.
  6. Add the action Speak Text (or Show Result if you prefer reading).
  7. Name the shortcut with the phrase you want to say, then tap Done.

Auto-import iPhone Reminders → task list

Anything you say to Siri the normal way ("remind me to sign Jack's permission slip") can flow into the hub's task list automatically. One-time setup:

  1. In the Reminders app, create a list called Family Hub (optional but recommended — it keeps personal reminders private; you can also make it your default list in Settings → Reminders).
  2. New shortcut → add Find Reminders → filter: List is Family Hub, Is Not Completed.
  3. Add Repeat with Each. Inside the repeat:
    • Get Contents of URL — paste https://xbdklcxsvdbesquodzsu.supabase.co/functions/v1/api/api/siri/task-add?title= and insert the Repeat Item variable right after title=; set the same Authorization header as your other shortcuts.
    • Complete Reminder (Repeat Item) — moves it out of Reminders once it's in the hub.
  4. Name it Sync My Reminders → Done.
  5. Make it automatic: Shortcuts app → Automation tab → + → Time of Day → e.g. 6:00 AM, Daily → Run Immediately (no confirmation) → choose Sync My Reminders.

The hub skips anything already on your list, so the automation can run as often as you like — add a second automation at 6 PM if you want evening pickups too. You can also just say "Hey Siri, sync my reminders" anytime.

Notes

• Each person uses their own key — allowance answers are personal, and a parent can revoke any phone's key in Admin without touching the others.
• The key never expires; treat it like a house key.
• Works from Apple Watch and CarPlay too, since Shortcuts sync across devices.
